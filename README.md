# Indoor-Air-Pollution-Detector
Indoor Air Pollution - Machine Learning Project

This project is about checking indoor air quality using machine learning.
It uses simple models to understand pollution levels and say if air is good or bad.

What this project does
We use data like CO, CO2, PM2.5, temperature and humidity.
The model learns from this data.
Then it predicts air quality.

Files in this project

dataset.csv
This file has the air data.

linear_regression.py
This predicts CO level.
It gives output using MSE.

logistic_regression.py
This tells if air is good or bad.

decision_tree.py
This uses tree logic to classify air quality.

random_forest.py
This is more accurate.
It uses many trees and gives better result.

Why this project

Indoor air pollution is a real problem.
This project helps to detect it early.
It can be used with sensors in future.
